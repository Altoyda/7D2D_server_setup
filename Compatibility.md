##  Compatibility

7 Days to Die will run on popular distros as long as the minimum requirements are met.

###  Server

For consistant uptime it is recomended you host a server in a data center. If you dont already have a server, you can rent a dedicated or vitual server from one of our sponsored providers, [OVHCloud](https://us.ovhcloud.com/), [Linode](https://linuxgsm.com/linode), [Vultr.](https://linuxgsm.com/vultr)

If the command-line isnt for your you can use [GameServerApp](https://www.gameserverapp.com/linuxgsm-support/?camp=linuxgsm&grp=sdtdserver) web-based game server management service.

###  Minimum Recommended Distros

-   ![Ubuntu Icon](https://linuxgsm.com/wp-content/themes/linuxgsm/img/ubuntu32.png) [Ubuntu 18.04 LTS](https://docs.linuxgsm.com/linux/distro#ubuntu)
-   ![Debian Icon](https://linuxgsm.com/wp-content/themes/linuxgsm/img/debian32.png) [Debian 10](https://docs.linuxgsm.com/linux/distro#debian)
-   ![Centos Icon](https://linuxgsm.com/wp-content/themes/linuxgsm/img/centos32.png) [CentOS 8](https://docs.linuxgsm.com/linux/distro#centos)

###  Other Distros

Although not tested, any distro with [tmux => 1.6](https://docs.linuxgsm.com/requirements/tmux) and [glibc => 2.27](https://docs.linuxgsm.com/requirements/glibc) should also be compatible with 7 Days to Die servers. For distros not listed above LinuxGSM does not check dependencies and should be manually checked.

###  Game Server Binary

Only 64-Bit game server binary is available.

##  SteamCMD

 [SteamCMD](https://docs.linuxgsm.com/steamcmd) by Valve is used to install and update this game server.
